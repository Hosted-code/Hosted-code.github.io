  • 查看本地有哪些包
  • 装完有可能出现如下提示, 按照提示输入命令执行下
  • 是否一定要照着 npm 或 GitHub 的建议來更新 package 似乎是不一定，而且得要靠个人判断，目前认为有效的判断方法是：透过在 GitHub 上的 issue 讨论来获得资讯。

    1. npm audit fix
    2. npm audit
    3. open More info
    4. cd Path && npm -i --save package@version
